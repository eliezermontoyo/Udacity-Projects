Intro to Programming / Stage 1: Make your Web Page

Session 1: The Basics of the Web and HTML

On this lesson I learned how the WWW works and what it is composed of (HTTP, Servers, Internet and Browser). Then I learned a little about HTML, some basic stuff like what it is and the difference between what you see in a browser and the actual code. I also saw that the HTML Markup is composed of elements like tags and attributes and that there are some tags you can close and other that don't need to be closed (a,img,em,src,ect.). Then we went over the differences between inline and block elements.

Session 2: Creating a Structured Document with HTML

On this lesson I learned how blocks in HTML work, how you can see the different blocks that make up a HTML document using developer tools in Google Chrome and that these blocks form an invisible rectangular box around each element. Also I learned about the DOM or the "Document Object Model" and how it is used to make a HTML into a tree-like structured document. Also this lesson explains that HTML and CSS are both languages used to create this structure. I also got introduced to various text editor tools like; Sublime Editor, Codepen and Scratchpad.

Session 3: Adding CSS Style to HTML Structure

This lesson was my favorite! I learned how to add style to my HTML document using CSS or "Cascading Style Sheets" and how to apply and edit them. I saw how important it is to avoid repetition by using inheritance in css and how avoiding needless repetion makes it easier to code and style. I learned how to use different resources to find all the different types of properties used in CSS and also learned how to use the "class" attribute to link a style to an element. Then I saw how the box model works and how it affects different elements in the HTML. At the end of this lesson I learned how to use HTML with CSS and how to refine the basic properties used to make this HTML document.
