Intro to Programming / Stage 2: Automate your page



Session 1: Introduction to "Serious" Programming

On this lesson I learned the difference between machines and computers and how computers can be programmed to do anything. These programs work by breaking problems into smaller pieces and then building steps to solve each piece. The example given to us in this session was with the programming language Python, which is used to tell a computer what to do. Python is executed with a Python Interpreter which is another program that runs the written code and converts it into a command that the computer understands.

In Python you can't just write anything and it will run. In Python you need to write everything with the correct Grammar, if not the Python Interpreter won't run the written code. A Python Expression is a way to organize your code so that the Python Interpreter can understand your Grammar.



Session 2: Variables and Strings

On this lesson I was introduced to the Variable, which is a way to assign a name to different stuff in Python, like strings or expressions. For example: for the string "hammer, nails, and glue" I can assign the variable Tools -> Tools = "hammer,nails and glue". Since I'm not a serious programmer yet, I can't say for certain what are the real uses for this, but I believe it is used to make really long expressions or strings into something more manageable. When assigning a variable, the equals sign does not have the same function as in a mathematical equation, instead it just gives a name to a value. So the equals sign in 1-1=0 is not the same as food="cookies".

I also learned about Strings which is a list of characters in order, for example "Wikepedia". For the name Wikipedia to be a string it has to be enclosed by quotes, which can be single (' '), double (""), or triple(''' '''). Strings and regular expressions are different ways to write code on Python, for example, it is not the same to write \\print 1+1+1\\ and \\print '1'+'1'+'1'\\, the answers are different. The first one will add the numbers 1 and the second one will combine all the number 1s to print '111'. Also in this lesson I learned how to find a specific character in a string with the function .find. This function will print out the position in the string of whatever character your looking for and also you can use it to find the occurence of that character, for example, if I give a value to the variable Eli (Eli = "A nice guy"), I can use the .find function to find in what position does the word "nice" starts; print Eli.find ("nice") the answer would be position 2, because it is the first occurence of the word nice in my string.



Session 3: Input -> Function -> Output

On this lesson I learned what is a function or as Dave calls them "procedures". A function is an expression used in python that starts by defining inputs like numbers or strings, then it produces an output. The first part of a function is the input, usually starting with the keyword "ref", then the function name and last a parameter or parameters. Example: ref name(Parameter):. After the input you have to define the function output. In this lesson we practiced with the function output "return". Example: return parameter + "s": Then you could use print to see the result of you function. Example: print name("ball") the program would give back "balls".

Functions are very useful because they help avoid repetitions by reusing them without having to write them again and you can use them with different parameters. It is important to always have a return statement in your functions. If you dont, the interpreter wont produce an output.
